📖 About

This website connects to IRIS Data Management Center's SeedLink server to receive real-time seismic waveform data. The data is transmitted using the SeedLink protocol and displayed in your browser using seisplotjs.

SeedLink Server: rtserve.iris.washington.edu:18000
Protocol: SeedLink v3
Data Format: Mini-SEED

For more information about IRIS SeedLink, visit: IRIS SeedLink Documentation

Tip: Try stations like: IU_KONO, GE_WLF, MN_AQU, US_TUC
